# Decision Helper

## Descripción del proyecto
Se Trata de una aplicación web pensada para ayudar a los usuarios a tomar decisiones comparando varias opciones según distintos criterios. El usuario crea una decisión, añade diferentes alternativas y evalua con puntuaciones para obtener una recomendación final.

## Problema que resuelve
Muchas veces tomar una decisión entre varias opciones resulta difícil porque intervienen varios factores a la vez. Esta aplicación organiza la información de manera clara y permite comparar opciones de forma más objetiva.

## Usuario objetivo
El usuario objetivo es cualquier persona que necesite comparar varias alternativas antes de tomar una decisión. Por ejemplo, estudiantes que comparan portátiles, personas que quieren elegir un curso, una película, un destino de viaje etc.

## Objetivos del proyecto
- Crear una aplicación web fullstack con frontend y backend separados
- Aplicar React, TypeScript y Tailwind en el frontend
- Construir una API con Node.js y Express
- Organizar la aplicación con una estructura clara y escalable
- Documentar el proceso de trabajo como en un proyecto profesional

## Funcionalidades principales
- Crear una decisión
- Editar y eliminar decisiones
- Añadir opciones a una decisión
- Editar y eliminar opciones
- Añadir criterios de evaluación
- Asignar un peso a cada criterio
- Puntuar cada opción según cada criterio
- Calcular un resultado final ponderado
- Mostrar la mejor opción recomendada

## Funcionalidades opcionales
- Filtrar decisiones por categoría
- Duplicar decisiones
- Añadir notas personales
- Marcar decisiones como finalizadas
- Exportar el resultado

## Tecnologías previstas
### Frontend
- React
- TypeScript
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express

## Estructura general de la aplicación
La aplicación tendrá un frontend donde el usuario podrá crear y gestionar decisiones, opciones y criterios. El backend ofrecerá una API REST para guardar y consultar los datos. La lógica principal del proyecto consistirá en calcular una puntuación final para cada opción según los criterios y sus pesos.

Decision Helper es una propuesta sencilla pero útil y con una utilidad clara para el usuario.
